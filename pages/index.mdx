# House of Nouns API

Welcome to the House of Nouns API.

## What is House of Nouns?

[House of Nouns](https://houseofnouns.wtf/) is a platform for [Nouns DAO](https://nouns.wtf/) governance. We were funded by both Prop House and an onchain proposal to create governance infrastructure for Nouns.

Our API includes data for proposals, votes, profiles, discussion, and more.
